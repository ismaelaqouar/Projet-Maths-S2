# Projet de Mathématiques - Résolution d'une équation différentielle d'ordre deux

Ce dépôt regroupe le travail de notre équipe concernant le **Sujet 1** du projet de mathématiques. 

L'objectif de ce projet est l'étude théorique et la résolution numérique d'un front d'onde stationnaire associé à une équation différentielle ordinaire du second ordre.

## Organisation du dépôt

Notre travail est structuré dans les dossiers suivants :

* **`/codes`** : Ce dossier contient notre script Python. Il inclut l'algorithme de détection des paramètres de l'onde stationnaire ainsi que l'implémentation du solveur numérique permettant de tracer la trajectoire.
* **`/rendu`** : Ce dossier contient notre rapport de projet final au format PDF. Il détaille l'état de l'art, les démonstrations de la partie analytique et de synthèse, ainsi que l'explication de notre démarche algorithmique.
